### Bee-crontab
[![GoDoc](https://godoc.org/github.com/sinksmell/bee-crontab?status.svg)](https://godoc.org/github.com/sinksmell/bee-crontab)

> 基于go语言实现的一个分布式的任务调度系统,可以取代crontab

* 可视化Web后台管理,配置定时任务方便
* 分布式架构,集群化调度,可用性高
* 秒级调度,准确性高
* 追踪任务执行状态,采集任务输出
* 可视化日志查看
